Turing Machine

The Turing machine is a theoretical mathematical model of computation . It describes an abstract machine . That has the capabilities to manipulate symbols on a strip of tape in relation to a table of rules. Which was invented in 1936. Alan would come to call it an automatic machine.

It consists of a long tape, which is divided into squares. Each one containing a symbol. This tape consists of infinite cells where each cell contains an input symbol or a special symbol called a blank. There's also a head pointer that points to the cell being read. All the components allow this machine, to read, write one step left or right.


How the Turing Machine Works:

  1. Initial Setup:

    2. The machine starts in an initial state (Start) with the head positioned facing the first square of the tape. The tape contains some input data, and the rest of the tape may be blank. The machine will start on first symbol on the tape.

  2. Reading the Symbol:

    3. The machine then reads the symbol currently under the head. Depending on the symbol and it's current state, the machine will be told it's next instruction.

  3. Writing a Symbol:

    4. Based on the transition rules, the machine might write a new symbol on the tape. Whilst if no rules apply the symbol will be left the same.

  4. Moving the Head:

    5. The transition function also tells the machine whether to move the head left or right by one square on the tape. This movement allows the machine to process the next part of the input.

  5. Changing the State:

    6. After reading, writing, and moving, the machine will move to the next state, specified by the transition function. The state determines the next action of the machine.

  6. Repeat:

    7. It continues to read symbols, write new symbols, move the head, and transition between states until it reaches a halt state. The halt state signifies that the machine has completed the computation.
